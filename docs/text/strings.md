# 문자열

문자열은 글자 여러 개를 담는 값입니다.

```koa
시작
    출력 "안녕하세요"
끝
```

## 변수에 담기

```koa
시작
    변수 name: 문자열 = "SKKOA"
    출력 name
끝
```

## 문자열 붙이기

```koa
시작
    변수 name: 문자열 = "SKKOA"
    출력 "Hello, " + name
끝
```

## 자주 쓰는 함수

```koa
시작
    변수 name: 문자열 = "SKKOA"

    출력 길이(name)
    출력 비교(name, "SKKOA")
    출력 부분문자열(name, 0, 2)
끝
```

- `길이(text)`: 글자 수를 알려줍니다.
- `비교(a, b)`: 같으면 `0`을 돌려줍니다.
- `부분문자열(text, start, length)`: 문자열 일부를 꺼냅니다.
