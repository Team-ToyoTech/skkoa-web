# 문자열

문자열(string)은 글자들의 모음이다.

## 현재 지원

문자열 리터럴 출력:

```koa
시작
    출력 "안녕하세요"
끝
```

문자열 변수:

```koa
변수 name: 문자열 = "SKKOA"
출력 name
```

문자열 결합:

```koa
출력 "Hello, " + name
```

문자열 입력과 기본 문자열 함수도 현재 지원한다.

```koa
시작
    변수 name: 문자열
    입력 name

    출력 길이(name)
    출력 비교(name, "SKKOA")
    출력 부분문자열(name, 0, 2)
끝
```

- `길이(text)`: 문자열 길이를 정수(integer)로 반환한다.
- `비교(a, b)`: C 표준 라이브러리의 `strcmp`처럼 같으면 `0`을 반환한다.
- `부분문자열(text, start, length)`: 시작 위치와 길이로 새 문자열을 만든다.
