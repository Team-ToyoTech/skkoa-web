# 배열

배열은 같은 종류의 값을 여러 개 담는 상자입니다.

```koa
변수 numbers: 정수[3]
```

위 코드는 정수 3개를 담을 수 있는 배열을 만듭니다.

## 값 넣기와 꺼내기

배열은 `0`번부터 시작합니다.

```koa
시작
    변수 numbers: 정수[3]

    numbers[0] = 10
    numbers[1] = 20
    numbers[2] = 30

    출력 numbers[0] + numbers[1] + numbers[2]
끝
```

## 처음부터 값 채우기

```koa
시작
    변수 numbers: 정수[] = [1, 2, 3, 4]
    출력 numbers[0]
끝
```

`정수[]`처럼 크기를 비우면 오른쪽 값 개수로 크기가 정해집니다.

## 기억할 점

`정수[3]` 배열에서 사용할 수 있는 번호는 `0`, `1`, `2`입니다.
