# 배열

배열(array)은 같은 타입의 값을 번호(index)로 저장하는 자료구조이다.

## 선언

```koa
변수 numbers: 정수[3]
```

현재 컴파일러는 고정 크기 배열을 지원한다. 원소 타입은 `정수`, `실수`, `논리`, `문자`, `문자열`, `포인터`를 사용할 수 있다.

## 대입과 접근

```koa
시작
    변수 numbers: 정수[3]

    numbers[0] = 10
    numbers[1] = 20
    numbers[2] = 30

    출력 numbers[0] + numbers[1] + numbers[2]
끝
```

## 배열 리터럴

배열 리터럴(array literal)은 선언과 재대입에서 사용할 수 있다.

```koa
변수 numbers: 정수[] = [1, 2, 3, 4, 5]
numbers = [5, 4, 3, 2, 1]
```

`정수[]`처럼 크기를 비워 두면 초기값 개수로 배열 크기를 정한다. 이미 선언된 배열에 다시 대입할 때는 원래 배열 크기와 리터럴 원소 개수가 같아야 한다.

## 인덱스 오류

컴파일러는 정적 숫자 인덱스가 배열 크기를 벗어날 가능성이 있으면 경고를 출력한다.
