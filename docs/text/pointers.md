# 포인터

포인터(pointer)는 어떤 값이 저장된 메모리 주소(memory address)를 담는 값이다.

포인터는 초보자에게 어렵지만, 컴퓨터 구조와 메모리 모델을 이해하는 데 중요하다.

## 설계 문법

```koa
변수 x: 정수 = 10
변수 p: 포인터<정수> = 주소(x)

출력 값(p)
```

의미:

- `주소(x)`: 변수 `x`의 메모리 주소를 얻는다.
- `값(p)`: 포인터 `p`가 가리키는 값을 읽는다.

## 현재 상태

기본 포인터 읽기는 현재 지원한다.

```koa
시작
    변수 x: 정수 = 10
    변수 p: 포인터<정수> = 주소(x)
    출력 값(p)
끝
```

`값(p) = 20`처럼 포인터가 가리키는 위치에 쓰는 문법은 아직 예정 기능이다.

## 향후 계획

- 지역 변수 주소 얻기
- 포인터 역참조(dereference)
- 포인터 쓰기
- 배열과 포인터 관계 설명
