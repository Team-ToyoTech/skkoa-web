# 포인터

포인터(pointer)는 어떤 값이 저장된 메모리 주소(memory address)를 담는 값이다.

포인터는 초보자에게 어렵지만, 컴퓨터 구조와 메모리 모델을 이해하는 데 중요하다.

## 설계 문법

```koa
변수 x: 정수 = 10
변수 p: 포인터<정수> = 주소(x)

출력 값(p)
```

의미:

- `주소(x)`: 변수 `x`의 메모리 주소를 얻는다.
- `값(p)`: 포인터 `p`가 가리키는 값을 읽는다.

## 현재 상태

포인터 읽기와 쓰기는 현재 지원한다.

```koa
시작
    변수 x: 정수 = 10
    변수 p: 포인터<정수> = 주소(x)
    출력 값(p)

    값(p) = 20
    출력 x
끝
```

동적 메모리(dynamic memory)는 `할당`과 `해제`로 사용할 수 있다.

```koa
시작
    변수 p: 포인터<정수> = 할당(8)
    값(p) = 42
    출력 값(p)
    해제(p)
끝
```

현재 포인터 연산은 교육용 최소 모델이다. 포인터 산술(pointer arithmetic)과 구조체 필드 주소 얻기는 향후 확장 범위이다.
