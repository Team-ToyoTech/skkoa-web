# 포인터

포인터는 처음 배울 때 어려울 수 있습니다. 변수, 조건문, 반복문, 함수, 배열을 먼저 익힌 뒤 읽어도 됩니다.

포인터는 어떤 값이 있는 위치를 담습니다.

```koa
시작
    변수 x: 정수 = 10
    변수 p: 포인터<정수> = 주소(x)

    출력 값(p)
끝
```

## 쓰는 말

- `주소(x)`: `x`가 있는 위치를 얻습니다.
- `값(p)`: `p`가 가리키는 위치의 값을 읽습니다.

## 값 바꾸기

```koa
시작
    변수 x: 정수 = 10
    변수 p: 포인터<정수> = 주소(x)

    값(p) = 25
    출력 x
끝
```

포인터는 고급 주제입니다. 내부 메모리 구조나 자세한 구현 내용은 README에 따로 정리되어 있습니다.
