# 제어 흐름

제어 흐름은 코드가 실행되는 순서입니다.

## 만약

조건이 맞을 때만 실행합니다.

```koa
시작
    변수 score: 정수 = 85

    만약 score >= 80 이면
        출력 "통과"
    아니면
        출력 "실패"
    끝
끝
```

## 아니면만약

조건을 여러 개 확인할 수 있습니다.

```koa
만약 score >= 90 이면
    출력 "A"
아니면만약 score >= 80 이면
    출력 "B"
아니면
    출력 "C"
끝
```

## 동안

조건이 맞는 동안 반복합니다.

```koa
시작
    변수 i: 정수 = 0

    동안 i < 5 반복
        출력 i
        i = i + 1
    끝
끝
```

## 반복

정해진 범위만큼 반복합니다.

```koa
시작
    반복 i: 1부터 3까지
        출력 i
    끝
끝
```
