# 제어 흐름

제어 흐름(control flow)은 프로그램이 어떤 순서로 실행되는지를 정한다.

## 조건문

```koa
시작
    변수 score: 정수 = 85

    만약 score >= 80 이면
        출력 "통과"
    아니면
        출력 "실패"
    끝
끝
```

다중 조건:

```koa
만약 score >= 90 이면
    출력 "A"
아니면만약 score >= 80 이면
    출력 "B"
아니면
    출력 "C"
끝
```

## 동안 반복

```koa
시작
    변수 i: 정수 = 0

    동안 i < 5 반복
        출력 i
        i = i + 1
    끝
끝
```

## 횟수 반복

```koa
반복 i: 0부터 10까지
    출력 i
끝
```

횟수 반복은 시작값부터 끝값까지 1씩 증가하며, 끝값을 포함한다.
