# 변수와 상수

변수(variable)는 값을 저장하는 이름 있는 공간이다.

## 변수 선언

표준 문법:

```koa
변수 이름: 자료형 = 값
```

예:

```koa
시작
    변수 x: 정수 = 10
    변수 y: 정수 = 20
    출력 x + y
끝
```

초기값은 생략할 수 있다. 생략하면 현재 컴파일러는 숫자와 포인터 계열은 `0`, 구조체 필드는 `0`으로 초기화한다. 문자열 변수는 출력 전에 문자열을 대입하거나 `입력`으로 값을 넣는 것을 권장한다.

```koa
변수 count: 정수
```

## 상수

상수(constant)는 한 번 정하면 다시 바꿀 수 없는 값이다.

```koa
상수 limit: 정수 = 100
```

현재 컴파일러는 상수에 다시 대입하면 오류를 출력한다.

## 대입

```koa
x = x + 1
```

대입(assignment)은 오른쪽 표현식(expression)을 계산한 뒤 왼쪽 변수에 저장한다.

## 오류 예

```koa
출력 missing
```

`missing`이 선언되지 않았다면 컴파일러는 선언되지 않은 변수 오류를 낸다.
