# 변수와 상수

변수는 값을 담는 이름입니다.

```koa
변수 x: 정수 = 10
```

위 코드는 `x`라는 이름에 숫자 `10`을 담습니다.

## 값 바꾸기

변수는 나중에 값을 바꿀 수 있습니다.

```koa
시작
    변수 score: 정수 = 70
    score = score + 10
    출력 score
끝
```

## 처음 값을 나중에 넣기

```koa
시작
    변수 age: 정수
    입력 age
    출력 age
끝
```

## 상수

상수는 한 번 정하면 바꾸지 않는 값입니다.

```koa
상수 maxScore: 정수 = 100
```

점수의 만점처럼 바뀌면 안 되는 값에 사용합니다.
