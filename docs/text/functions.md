# 함수

함수는 자주 쓰는 코드를 이름으로 묶은 것입니다.

## 값을 돌려주는 함수

```koa
함수 더하기(a: 정수, b: 정수): 정수
    반환 a + b
끝

시작
    출력 더하기(3, 4)
끝
```

`a`와 `b`는 함수가 받는 값입니다. `반환`은 결과를 돌려줍니다.

## 값을 돌려주지 않는 함수

```koa
함수 인사(): 없음
    출력 "안녕하세요"
끝

시작
    인사()
끝
```

## 구조체를 바꾸는 함수

구조체를 함수에 넣으면 함수 안에서 필드를 바꿀 수 있습니다.

```koa
구조체 사람
    age: 정수
끝

함수 한살더하기(user: 사람): 없음
    user.age = user.age + 1
끝

시작
    변수 user: 사람
    user.age = 10
    한살더하기(user)
    출력 user.age
끝
```
