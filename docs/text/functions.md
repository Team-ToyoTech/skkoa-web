# 함수

함수(function)는 이름이 있는 코드 묶음이다. 값을 받을 수 있고, 값을 반환(return)할 수 있다.

## 정수 반환 함수

```koa
함수 더하기(a: 정수, b: 정수): 정수
    반환 a + b
끝

시작
    변수 result: 정수 = 더하기(3, 4)
    출력 result
끝
```

## 반환값이 없는 함수

```koa
함수 인사(): 없음
    출력 "안녕하세요"
끝

시작
    인사()
끝
```

`없음(void)` 반환 타입은 현재 지원한다. 반환값이 없는 함수는 `인사()`처럼 한 줄 호출문으로 실행한다.

## 현재 제한

- 매개변수(parameter)는 Windows에서 최대 4개, Linux/macOS x86-64에서 최대 6개까지 지원한다.
- 매개변수 타입은 `정수`, `논리`를 우선 지원한다.
- 배열 매개변수, 문자열 매개변수, 포인터 매개변수는 예정 기능이다.
