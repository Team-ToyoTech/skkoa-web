# 문법

## 파일 구조

SKKOA 프로그램은 하나의 `시작` 블록을 가진다.

```koa
시작
    출력 "Hello"
끝
```

함수(function)는 `시작` 블록 앞에 정의할 수 있다.

```koa
함수 더하기(a: 정수, b: 정수): 정수
    반환 a + b
끝

시작
    출력 더하기(3, 4)
끝
```

## 주석

한 줄 주석(comment):

```koa
# 한 줄 주석
// 한 줄 주석
```

여러 줄 주석:

```koa
/*
여러 줄 주석
*/
```

## 키워드

```text
시작 끝 변수 상수 출력 입력 만약 이면 아니면 아니면만약
동안 반복 함수 반환 정수 실수 논리 문자 문자열 없음
참 거짓 그리고 또는 아님 주소 값 포인터
```

## 현재 지원 상태

**현재 지원**: 시작/끝, 변수, 상수, 출력, 입력, 조건문, 동안 반복, 횟수 반복, 함수, 정수 배열, 문자열 결합, 기본 포인터 읽기.

**설계됨 / 예정**: 포인터 쓰기, 구조체, 모듈 시스템.
