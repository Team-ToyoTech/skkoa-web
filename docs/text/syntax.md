# 문법

## 기본 모양

모든 프로그램은 보통 `시작`과 `끝` 사이에 코드를 씁니다.

```koa
시작
    출력 "Hello"
끝
```

## 줄마다 하나씩 쓰기

한 줄에 하나의 일을 쓰면 읽기 쉽습니다.

```koa
변수 age: 정수 = 20
출력 age
```

## 주석

주석은 컴퓨터가 실행하지 않는 설명입니다.

```koa
# 이름을 출력합니다
출력 "SKKOA"
```

## 다른 파일 가져오기

다른 `.koa` 파일이나 설치된 표준 모듈을 가져올 수 있습니다.

```koa
가져오기 "stack.koa"
가져오기 "queue.koa"
```

가져온 뒤에는 그 파일 안에 있는 구조체와 함수를 바로 사용할 수 있습니다.

## 함수 만들기

```koa
함수 더하기(a: 정수, b: 정수): 정수
    반환 a + b
끝

시작
    출력 더하기(3, 4)
끝
```

## 구조체 만들기

구조체는 여러 값을 한 이름으로 묶을 때 사용합니다.

```koa
구조체 사람
    name: 문자열
    age: 정수
끝
```
