# 문법

## 파일 구조

SKKOA 프로그램은 하나의 `시작` 블록을 가진다. `구조체`와 `함수`는 `시작` 블록 앞에 정의할 수 있다.

```koa
시작
    출력 "Hello"
끝
```

모듈(module)은 다른 `.koa` 파일을 현재 파일 앞에 붙이는 방식으로 가져온다.

```koa
가져오기 "math_module.koa"
```

구조체(structure)는 이름 있는 필드(field)를 묶는다.

```koa
구조체 사람
    name: 문자열
    age: 정수
끝
```

함수(function)는 `시작` 블록 앞에 정의할 수 있다.

```koa
함수 더하기(a: 정수, b: 정수): 정수
    반환 a + b
끝

시작
    출력 더하기(3, 4)
끝
```

## 주석

한 줄 주석(comment):

```koa
# 한 줄 주석
// 한 줄 주석
```

여러 줄 주석:

```koa
/*
여러 줄 주석
*/
```

## 현재 지원 상태

**현재 지원**: 시작/끝, 변수, 상수, 출력, 입력, 조건문, 동안 반복, 횟수 반복, 함수, 배열, 배열 리터럴, 문자열 결합, 포인터 읽기/쓰기, 구조체, 모듈 가져오기.

**확장 예정**: 구조체 리터럴, 구조체 값 반환, 네임스페이스가 있는 모듈 시스템.
