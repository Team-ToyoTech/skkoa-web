# 표준 라이브러리

표준 라이브러리는 자주 쓰는 기능을 미리 준비해 둔 것입니다.

## 바로 쓸 수 있는 기능

```koa
출력 "안녕하세요"
입력 age
```

문자열 함수:

```koa
출력 길이("SKKOA")
출력 비교("A", "B")
출력 부분문자열("SKKOA", 0, 2)
```

메모리 함수:

```koa
변수 p: 포인터<정수> = 할당(8)
해제(p)
```

## 가져와서 쓰는 모듈

스택:

```koa
가져오기 "stack.koa"
```

큐:

```koa
가져오기 "queue.koa"
```

스택과 큐를 모두 가져오기:

```koa
가져오기 "structures.koa"
```

설치 파일로 SKKOA를 설치했다면 위 모듈은 함께 설치됩니다.
