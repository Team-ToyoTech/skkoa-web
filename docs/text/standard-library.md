# 표준 라이브러리

표준 라이브러리(standard library)는 자주 쓰는 기능을 미리 제공하는 코드 모음이다.

## 현재 지원

현재 컴파일러는 언어 내장 출력문과 정수/논리/실수/문자 입력문을 제공한다.

```koa
출력 "안녕하세요"
출력 10
출력 x + y
```

입력:

```koa
변수 age: 정수
입력 age
```

내부적으로는 생성된 NASM 어셈블리가 C 표준 라이브러리의 `printf`와 `scanf`를 호출한다.

## 설계됨 / 예정

메모리:

```koa
변수 p: 포인터<정수> = 할당(8)
해제(p)
```

자료구조:

- 스택(Stack)
- 큐(Queue)
- 리스트(List)
- 맵(Map)

문자열:

- 길이
- 결합
- 비교
- 부분 문자열
