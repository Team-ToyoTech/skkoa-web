# 문법 요약

자주 쓰는 모양만 모아 둔 빠른 참고표입니다.

## 기본

```koa
시작
    출력 "안녕하세요"
끝
```

## 변수

```koa
변수 이름: 자료형 = 값
상수 이름: 자료형 = 값
```

## 조건

```koa
만약 조건 이면
    실행할 코드
아니면
    다른 코드
끝
```

## 반복

```koa
동안 조건 반복
    실행할 코드
끝
```

```koa
반복 i: 1부터 3까지
    출력 i
끝
```

## 함수

```koa
함수 이름(a: 정수, b: 정수): 정수
    반환 a + b
끝
```

## 배열

```koa
변수 numbers: 정수[3]
numbers[0] = 10
```

## 구조체

```koa
구조체 사람
    age: 정수
끝

변수 user: 사람
user.age = 10
```

## 모듈

```koa
가져오기 "stack.koa"
가져오기 "queue.koa"
```

가져온 모듈의 함수와 구조체는 바로 사용할 수 있습니다.
