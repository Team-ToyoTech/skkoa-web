# 자료형

자료형(type)은 값이 어떤 종류인지 알려주는 이름이다.

| SKKOA | 영어 | 현재 상태 |
| --- | --- | --- |
| `정수` | integer | 현재 지원 |
| `논리` | boolean | 현재 지원 |
| `문자열` | string | 현재 지원 |
| `실수` | floating point number | 현재 지원 |
| `문자` | character | ASCII 문자 리터럴 현재 지원 |
| `없음` | void | 함수 반환 타입으로 현재 지원 |
| `포인터<정수>` | pointer | 주소 얻기, 읽기, 쓰기, 동적 할당 현재 지원 |
| 사용자 정의 구조체 | structure | 현재 지원 |

## 논리 값

논리(boolean)는 참과 거짓을 표현한다.

```koa
변수 passed: 논리 = 참
```

현재 컴파일러는 논리 값을 내부적으로 `1`과 `0`으로 다룬다.

## 문자열

```koa
출력 "문자열 리터럴"
```

현재 컴파일러는 문자열 리터럴(string literal), 문자열 변수, 문자열 입력, 문자열 결합, `길이`, `비교`, `부분문자열`을 지원한다.

## 구조체

구조체(structure)는 여러 필드를 하나의 타입으로 묶는다.

```koa
구조체 점
    x: 정수
    y: 정수
끝

시작
    변수 p: 점
    p.x = 10
    p.y = 20
끝
```
