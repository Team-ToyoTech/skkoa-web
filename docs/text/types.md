# 자료형

자료형은 값의 종류입니다.

| 자료형 | 담는 값 |
| --- | --- |
| `정수` | 1, 2, 100 같은 숫자 |
| `실수` | 1.5, 3.14 같은 소수 |
| `논리` | `참` 또는 `거짓` |
| `문자` | `'A'` 같은 한 글자 |
| `문자열` | `"안녕하세요"` 같은 글 |
| `없음` | 함수가 돌려주는 값이 없을 때 |
| `포인터<정수>` | 주소를 배울 때 사용하는 값 |

## 예시

```koa
시작
    변수 age: 정수 = 20
    변수 height: 실수 = 170.5
    변수 passed: 논리 = 참
    변수 name: 문자열 = "SKKOA"

    출력 age
    출력 height
    출력 passed
    출력 name
끝
```

## 구조체 자료형

직접 만든 구조체도 자료형처럼 사용할 수 있습니다.

```koa
구조체 점
    x: 정수
    y: 정수
끝

시작
    변수 p: 점
    p.x = 10
    p.y = 20
끝
```
