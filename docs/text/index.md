# SKKOA 문서

SKKOA는 **Starter Kit with Korean Oriented Architecture**의 약자로, 한국어 기반 명령어와 직관적인 문법을 사용하는 교육용 프로그래밍 언어이다.

```koa
시작
    출력 "안녕하세요, SKKOA!"
끝
```

## 현재 지원

- `.koa` 파일 구조
- `시작` / `끝` 프로그램 블록
- `정수`, `실수`, `논리`, `문자`, `문자열`
- 변수(variable)와 상수(constant)
- 산술, 비교, 논리 연산
- 문자열 결합
- `만약`, `아니면만약`, `아니면`
- `동안` 반복
- `반복 i: 0부터 10까지`
- `입력`
- 여러 자료형과 배열 매개변수를 받는 함수(function)
- 고정 크기 배열(array), 배열 리터럴
- `주소(x)`, `값(p)`, `값(p) = ...` 포인터 읽기/쓰기
- `할당(크기)`, `해제(p)` 기본 동적 메모리 호출
- 구조체(structure) 변수와 필드 접근
- `가져오기 "파일.koa"` 모듈(module) 가져오기
- 문자열 표준 함수 `길이`, `비교`, `부분문자열`
- NASM x86-64 어셈블리 생성

## 확장 예정

- 스택(Stack), 큐(Queue), 맵(Map) 표준 라이브러리
- 구조체 리터럴과 구조체 값 반환
- 네임스페이스(namespace)를 가진 모듈 시스템

## 문서 읽는 순서

처음 읽는다면 **소개**, **시작하기**, **문법**, **자료형**, **변수와 상수**, **제어 흐름**, **함수** 순서로 읽는 것을 권장한다.
