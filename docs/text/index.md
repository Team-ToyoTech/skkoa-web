# SKKOA 문서

SKKOA는 **Starter Kit with Korean Oriented Architecture**의 약자로, 한국어 기반 명령어와 직관적인 문법을 사용하는 교육용 프로그래밍 언어이다.

```koa
시작
    출력 "안녕하세요, SKKOA!"
끝
```

## 현재 지원

- `.koa` 파일 구조
- `시작` / `끝` 프로그램 블록
- `정수`, `실수`, `논리`, `문자`, `문자열`
- 변수(variable)와 상수(constant)
- 산술, 비교, 논리 연산
- 문자열 결합
- `만약`, `아니면만약`, `아니면`
- `동안` 반복
- `반복 i: 0부터 10까지`
- `입력`
- 정수 반환 함수(function)
- 고정 크기 정수 배열(array)
- `주소(x)`, `값(p)` 기본 포인터 읽기
- `할당(크기)`, `해제(p)` 기본 동적 메모리 호출
- NASM x86-64 어셈블리 생성

## 설계됨 / 예정

- 포인터 쓰기
- 구조체(structure)
- 모듈(module) 시스템
- 확장 표준 라이브러리(standard library)

## 문서 읽는 순서

처음 읽는다면 **소개**, **시작하기**, **문법**, **자료형**, **변수와 상수**, **제어 흐름**, **함수** 순서로 읽는 것을 권장한다.
