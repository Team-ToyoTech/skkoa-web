# SKKOA 문서

SKKOA는 한국어 문법으로 배우는 교육용 프로그래밍 언어입니다.

처음에는 아래 모양만 기억하면 됩니다.

```koa
시작
    출력 "안녕하세요, SKKOA!"
끝
```

`시작`부터 프로그램이 실행되고, `끝`에서 프로그램 블록이 끝납니다.

## 무엇을 배울 수 있나요?

- 글자와 숫자 출력하기
- 값을 저장하는 변수 만들기
- `만약`으로 조건 나누기
- `동안`, `반복`으로 여러 번 실행하기
- 함수를 만들어 코드 묶기
- 배열로 여러 값을 한 번에 다루기
- `stack.koa`, `queue.koa`를 가져와 스택과 큐 사용하기

## 먼저 읽을 순서

1. 소개
2. 시작하기
3. 문법
4. 자료형
5. 변수와 상수
6. 제어 흐름
7. 배열
8. 자료구조

컴파일러 내부 구조, 빌드 방법, 구현 세부 사항은 저장소의 `README.md`와 `compiler/README.md`에 정리되어 있습니다.
