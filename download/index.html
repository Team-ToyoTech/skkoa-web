Windows용 SKKOA 컴파일러

Windows에서는 실행 가능한 설치 파일을 받아 SKKOA CLI를 설치합니다. 설치 후 새 터미널에서 skkoa hello.koa를 실행합니다.

Windows 다운로드
WIN

Windows

실행 파일 형식: .cmd

Windows 설치 파일 받기
MAC

macOS

실행 파일 형식: .command

macOS 설치 파일 받기
LIN

Linux

실행 파일 형식: .sh

Linux 설치 파일 받기

설치 후 사용

skkoa hello.koa
skkoa hello.koa -o hello
skkoa hello.koa --emit-asm

실행 파일 링크 단계에는 NASM과 GCC가 필요합니다. 설치 파일은 C++17 컴파일러로 SKKOA CLI를 빌드하고, PATH에서 사용할 수 있는 위치에 skkoa 명령을 설치합니다.
