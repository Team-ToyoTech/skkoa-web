Windows용 SKKOA 컴파일러

Windows에서는 통합 설치 파일 하나가 SKKOA CLI와 어셈블/링크 도구를 함께 설치합니다. 설치 후 새 터미널에서 skkoa hello.koa를 실행합니다.

Windows 다운로드
WIN

Windows

SKKOA CLI, MSYS2 GCC, NASM을 함께 설치합니다.

Windows 설치 파일 받기
MAC

macOS

Xcode 도구, Homebrew, NASM을 확인하고 설치합니다.

macOS 설치 파일 받기
LIN

Linux

배포판 패키지 관리자로 GCC와 NASM을 설치합니다.

Linux 설치 파일 받기

설치 후 사용

skkoa hello.koa
skkoa hello.koa -o hello
skkoa hello.koa --emit-asm

설치 파일은 SKKOA 컴파일러와 어셈블리 변환/링크에 필요한 도구를 함께 준비합니다. 별도로 C++ 컴파일러, NASM, GCC를 미리 설치하지 않아도 됩니다. 단, 도구 다운로드를 위한 인터넷 연결과 OS별 설치 권한은 필요합니다. 스택과 큐 표준 모듈도 함께 설치되므로 코드에서 가져오기 "stack.koa", 가져오기 "queue.koa"를 바로 사용할 수 있습니다.
